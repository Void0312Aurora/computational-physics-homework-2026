# LaTeX Report

This directory contains the classroom-report LaTeX version synchronized from `../answer.md`.
The original Markdown, DOCX, and PDF files in `docs/answer/` are kept unchanged.

## Build

```bash
make -C HW/02/docs/answer/latex
```

The generated PDF is:

```text
HW/02/docs/answer/latex/answer.pdf
```

## Typography

- A4 page with 2.5 cm margins.
- Small-four body text, first-line indentation, and 1.36 line stretch.
- Compact heading spacing, small table/figure captions, and framed small code blocks.
