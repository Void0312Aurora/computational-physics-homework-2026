# HW/02 Notes

## Scope

This directory contains independent per-problem C scripts for all five homework
problems:

- Gregory-Leibniz summation order
- Machin's formula for pi
- Endianness detection
- Machine epsilon detection
- A recurrence that should converge to pi but becomes unstable

## Why C here

The homework statements explicitly ask for C programs in Problems 3 and 4, and Problem 2 explicitly points to `quadmath.h`. The files under `scripts/` are intentionally standalone, so each submitted problem can be compiled and inspected without pulling in unrelated problem code. The top-level `Makefile` only orchestrates building, running, document export, and packaging.

## Files

- `scripts/problem1.c` ... `scripts/problem5.c`: standalone per-problem programs
- `Makefile`: build, run, and document-export entry point
- `result/temp-01.log`: raw output captured from the five standalone programs
- `docs/answer/answer.md`: summary report source
- `docs/answer/render_docs.py`: exports `docx` and `pdf`

## Build and run

```bash
cd HW/02
make
make run
```

To refresh the report outputs:

```bash
cd HW/02
make docs
```

To refresh the final submission folder and tar archive:

```bash
cd HW/02
make package
```
