# LaTeX Report

This directory contains the classroom-report LaTeX version aligned with the
indented `HW/00` style reference. The generated `answer.pdf` is also copied to
`HW/06/docs/answer/answer.pdf` and the submission package.

## Build

```bash
make -C HW/06/docs/answer/latex
```

The generated PDF is:

```text
HW/06/docs/answer/latex/answer.pdf
```

The canonical packaged PDF is:

```text
HW/06/docs/answer/answer.pdf
```

## Typography

- A4 page with 2.7 cm side margins and 2.5 cm top/bottom margins.
- Small-four body text, first-line indentation, and 1.36 line stretch.
- Compact heading spacing, small table/figure captions, and framed small code blocks.
